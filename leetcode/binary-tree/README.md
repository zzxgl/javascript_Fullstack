# 二叉树遍历 前序遍历 中序遍历 后序遍历
 - 前序遍历  根左右 先遍历根节点再遍历左子树 然后右子树  A B D G H C E I F
 - 中序遍历  左根右 先遍历左子树再遍历根节点 然后右子数  G D H B A E I C F
 - 后序遍历  左右根 先遍历左子树再遍历右子树 然后根子树  G H D B I E F C A
